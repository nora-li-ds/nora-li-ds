## Flagship Project

### Stablecoin Transfer Behaviour and Fee Regime Classification

This MSc dissertation project uses public Ethereum and Dune Analytics data to analyse stablecoin transfer behaviour under different gas-fee regimes.

The project builds a reproducible SQL/Python workflow to extract, clean and structure transaction-level data, with a focus on validation, robustness checks and careful interpretation.

The goal is not to label activity as illicit, but to explore whether fee-regime behaviour can provide transparent and auditable indicators that may support crypto-AML research and transaction-monitoring workflows.

Key themes:

- SQL/Python data pipeline
- stablecoin transfer analysis
- gas-fee regime classification
- lagged event windows
- validation and negative-control checks
- behavioural signal interpretation
- transparent documentation of assumptions and limitations
